Caso de estudio:

Diseño UX/UI y desarrollo Frontend

Contexto del proyecto

Desarrollo del sitio web para Sweens, marca de suavizante enfocada en fragancias y cuidado de la ropa.
El objetivo del proyecto fue crear una experiencia digital clara y visual que permitiera a los usuarios explorar las variedades de producto, conocer sus beneficios y acceder a contenido útil como tips de lavado.

Mi Rol

Participé en el proyecto como Frontend Developer y diseñadora UX/UI.

Responsabilidades principales:

  • Entrevista con usuarios
  • Research
  • Diseño de interfaz y estructura del sitio
  • Maquetación frontend del sitio web
  • Desarrollo de interfaces responsive
  • Adaptación del diseño a código
  • Optimización visual para campañas digitales

Investigación de usuarios

Para entender qué tipo de contenido era más relevante para los usuarios, se revisaron hábitos de consumo y comportamiento al elegir productos de limpieza.

Preguntas clave

"¿Qué buscan los usuarios al elegir un suavizante?"

"¿Qué información consideran importante antes de comprar?"

Hallazgos principales

  • Los usuarios valoran conocer las fragancias y beneficios del producto.
  • Existe interés en tips de lavado y cuidado de la ropa.
  • Las interfaces visuales y simples facilitan la exploración del contenido.

Análisis competitivo

Se revisaron sitios de marcas similares del rubro para identificar buenas prácticas en:

  • Presentación de productos
  • Jerarquía de información
  • Experiencia visual

Wireframes de baja calidad

Se desarrollaron wireframes iniciales de baja fidelidad para definir:

  • Estructura del sitio
  • Jerarquía de contenido
  • Distribución de secciones principales

Estos bocetos permitieron validar la organización de la información antes de pasar al diseño visual.

Diseños MVP

A partir de los wireframes se desarrolló el diseño visual del sitio. El diseño se enfocó en:

  • Destacar las fragancias y beneficios del producto
  • Mantener coherencia con la identidad de la marca
  • Experiencia visual

    • Se definieron:

  • Paleta de colores
  • Tipografía
  • Componentes visuales del sitio

Diseños MVP

El sitio fue desarrollado como una interfaz responsive, adaptada para distintos dispositivos.

Tecnologías utilizadas:

  • HTML
  • CSS
  • JavaScript

    • Durante el desarrollo se trabajó en:

  • Maquetación responsive del sitio
  • Implementación de componentes visuales
  • Optimización de imágenes y assets
  • Adaptación del diseño UX/UI a desarrollo frontend

Lanzamiento del sitio y los próximos pasos

Se envío una url de prueba a cliente para que pudieran revisar la estructura y que nos terminaran de confirmar la información, de los productos, los tips, etc. Y también para que revisaran en distintos dispositivos tanto web como mobile la estructura html del sitio.

Y como último paso antes de la salida a producción se solicitó que en el home se añadiera un background y unas nubes con animación, para que diera una sensación de "estar entre las nubes" y que no se viera tan vacío.
